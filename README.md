## 설치

```bash
git clone git@github.com:youjeonghan/claude-plugins.git ~/claude-plugins
ln -s ~/claude-plugins/plugins/self-evolve/0.1.0 ~/.claude/plugins/self-evolve
```

```bash
cd ~/claude-plugins && git pull
```
